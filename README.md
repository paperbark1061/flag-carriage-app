# FlagCarriage iOS App

SwiftUI app for controlling the flag carriage over WiFi.

## How to open in Xcode

1. In Xcode — **Clone Git Repository**
2. Paste: `https://github.com/paperbark1061/flag-carriage-app.git`
3. Open `FlagCarriage.xcodeproj`
4. Select your iPhone or simulator
5. Hit ⌘R
